#### 시점 되돌릴 때
- 커밋 후 작업 되돌리고 싶을 때

```bash
$ git revert hash값     # hash값: 되돌리고 싶은 시점 값
```

#### 경고 메세지 
- 줄바꿈(LF vs CRLF) 문제로 인한 Git 경고 메세지 나올 때

<p align="left">
  <img src="./img/lf_warning_msg.png" width="300"/>
</p>

```bash
$ git config --global core.autocrlf true
```

# Git 클론 후 작업할 때

원격 저장소에 있는 프로젝트를 로컬로 가져와서 작업할 때 사용합니다.

## 상황

이미 GitHub 등에 있는 프로젝트를 내 PC로 복제(clone)한 뒤, 같은 브랜치에서 작업을 이어갈 때입니다.

## 원격에서 로컬로 가져오기

```bash
git clone 원격-저장소-url
cd 복제한폴더명
```

연결이 안 되어 있다면:

```bash
git remote add origin 원격-저장소-url
git branch -m main
git pull origin main
git push -u origin main
```

## 참고

- `git clone`은 저장소를 복제해 오는 것이므로 `git init`은 필요 없습니다.
- 연결 확인: `git remote -v`

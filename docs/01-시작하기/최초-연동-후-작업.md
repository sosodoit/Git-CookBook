# Git 최초 연동 후 작업할 때

로컬에서 작업한 내용을 Git으로 처음 올릴 때 사용합니다.

## 상황

로컬 폴더에 이미 작업이 있고, 원격 저장소(GitHub 등)와 연결해 처음 올리려는 경우입니다.

## 최초 연동 시 Git 계정 설정

```bash
git config --global user.name "ID"
git config --global user.email "mail계정"
git config --list
```

## 로컬에서 원격으로 올리기

```bash
git init
git add .
git commit -m "message(필수)"
git branch -m main
git remote add origin 원격-저장소-url
git push -u origin main
```

## 참고

- `git add .`는 현재 폴더의 모든 변경 사항을 스테이징합니다. 작업 단위로 `git add 파일명` 후 커밋하는 것을 권장합니다.

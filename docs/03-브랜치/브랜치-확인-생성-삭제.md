# 브랜치 확인·생성·삭제

현재 브랜치 확인, 새 브랜치 생성, 브랜치 삭제 방법입니다.

## 상황

어디서 작업 중인지 확인하거나, 새 브랜치를 만들거나, 작업이 끝난 브랜치를 정리할 때 사용합니다.

## 브랜치 확인

```bash
git branch
```

- 현재 위치한 브랜치 앞에 `*`가 표시됩니다.

```bash
git branch -a
```

- 로컬·원격 포함 전체 브랜치 목록을 볼 수 있습니다.

```bash
git branch -r
```

- 원격 브랜치 목록만 볼 때 사용합니다.

## 새 브랜치 생성

```bash
git checkout -b 브랜치명
git push --set-upstream origin 브랜치명
```

## 브랜치 삭제

작업이 완료되고 병합이 끝난 브랜치는 정리하는 것이 좋습니다. 협업 시에는 병합 일자를 정한 뒤, 같은 시점에 모두 새 브랜치를 만드는 방식이 정리하기 쉽습니다.

**로컬 브랜치 삭제**

```bash
git branch -d 브랜치명
```

**원격 브랜치 삭제**

```bash
git push origin -d 브랜치명
```

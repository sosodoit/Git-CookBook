# 다 버리고 이전 시점으로 되돌리기

특정 커밋 이후 작업을 없애고, 그 시점 상태로 되돌리고 싶을 때 사용합니다.

## 상황

커밋한 내용을 되돌리고 싶을 때 `git revert`로 해당 커밋을 취소하는 방식입니다. (이력을 남기는 안전한 방법입니다.)

## 명령어

```bash
git revert 해시값
```

- `해시값`: 되돌리고 싶은 시점의 커밋 해시. `git log` 또는 `git reflog`로 확인할 수 있습니다.

## 참고

- `git revert`는 “해당 커밋을 없애는 새 커밋”을 만듭니다. 원격에 이미 푸시한 커밋을 되돌릴 때 적합합니다.
- 로컬에서만 되돌리고 이력 자체를 지우고 싶다면 `git reset`을 검토할 수 있으나, 이미 푸시한 브랜치에서는 협업자에게 영향이 있으므로 주의해 사용해야 합니다.
